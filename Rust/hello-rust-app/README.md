# Hello Rust App

This is a simple Rust application that prints "Hello, world!" to the console.

## Getting Started

To build and run this application, you need to have Rust installed on your machine. You can install Rust by following the instructions at [rust-lang.org](https://www.rust-lang.org/).

### Building the Application

Navigate to the project directory and run the following command:

```
cargo build
```

### Running the Application

After building the application, you can run it with the following command:

```
cargo run
```

You should see the output:

```
Hello, world!
```

## Project Structure

- `src/main.rs`: The entry point of the application.
- `.vscode/launch.json`: Debugging configuration.
- `.vscode/tasks.json`: Build and run tasks.
- `.gitignore`: Files and directories to ignore by Git.
- `Cargo.toml`: Project configuration file.
